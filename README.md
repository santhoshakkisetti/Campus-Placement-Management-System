# Smart Campus Placement Management System

## Overview

The Smart Campus Placement Management System is a custom ServiceNow application developed to automate the campus recruitment lifecycle, including job postings, student applications, interview scheduling, candidate selection, and placement analytics.

## Key Features

- Student Registration & Profile Management
- Company & Job Opening Management
- Placement Application Tracking
- Interview Scheduling Automation
- Role-Based Access Control (ACLs)
- Flow Designer Automation
- Reports & Dashboards

## Technologies Used

- ServiceNow
- App Engine Studio
- Flow Designer
- Service Catalog
- Record Producers
- ACLs
- Reports
- Dashboards
